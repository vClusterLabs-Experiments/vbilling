Configuration

All the knobs for tuning vBilling behavior: environment variables, Helm chart values, and Lago pricing setup.

Environment Variables

When running vBilling as a binary or in a container, the following environment variables control its behavior. All have sensible defaults except LAGO_API_KEY, which is required.

Variable Description Default
LAGO_API_URL Base URL of the Lago API server http://localhost:3000
LAGO_API_KEY Required. API key for authenticating with Lago (none)
COLLECTION_INTERVAL How often to scrape metrics and send usage events to Lago. Accepts Go duration strings (e.g. 60s, 5m) 60s
RECONCILE_INTERVAL How often to scan for new or removed vClusters and sync billing entities. Accepts Go duration strings 30s
DEFAULT_PLAN_CODE Lago plan code to assign to newly discovered vClusters vcluster-standard
BILLING_CURRENCY ISO 4217 currency code for all billing operations USD
PROMETHEUS_URL Prometheus server URL for DCGM GPU utilization and network egress metrics. Leave empty to disable (empty)
SPOT_DISCOUNT_PERCENT Percentage discount applied to workloads running on spot/preemptible nodes. A value of 60 means spot nodes cost 40% of on-demand 60
WATCH_NAMESPACES Comma-separated list of namespaces to monitor for vClusters. Empty means watch all namespaces (all namespaces)
VCLUSTER_NODE_LABEL Custom node label selector for private node detection. Format: key=value-pattern where %s is replaced with the vCluster name. See Private Nodes (none)
i
COLLECTION_INTERVAL directly affects billing granularity and accuracy. Shorter intervals produce more events but more precise billing. The default of 60 seconds is a good balance.

Helm Chart Values

When deploying via Helm, all configuration is specified through values.yaml or --set flags. The Helm chart translates these into the environment variables shown above.

Lago connection

ValueDescriptionDefault
lago.apiURL Lago API URL (in-cluster service address) http://lago-api.lago-system.svc.cluster.local:3000
lago.apiKey Lago API key (plaintext, use --set) (empty)
lago.existingSecret Name of an existing Kubernetes Secret containing the API key (key: api-key). Preferred over lago.apiKey for production (empty)

Billing settings

ValueDescriptionDefault
billing.defaultPlanCode Lago plan code for new vClusters vcluster-standard
billing.currency Billing currency (ISO 4217) USD
billing.collectionInterval Metrics scrape interval 60s
billing.reconcileInterval vCluster discovery interval 30s

Pricing defaults

ValueDescriptionDefault
pricing.cpuPerCoreHour Price per CPU core-hour 0.05
pricing.memoryPerGBHour Price per memory GB-hour 0.01
pricing.storagePerGBHour Price per storage GB-hour 0.001
pricing.instancePerHour Flat hourly charge per vCluster instance 0.10
pricing.gpuDefaultPerHour Default GPU price per hour (when type not in gpuPrices) 2.50
pricing.networkEgressPerGB Price per GB of network egress 0.05
pricing.lbPerHour Price per LoadBalancer service-hour 0.025
pricing.gpuPrices Comma-separated GPU type-specific pricing in TYPE:PRICE format NVIDIA-A100-SXM4-80GB:3.50,NVIDIA-H100-80GB-HBM3:5.00,NVIDIA-H100-SXM5-80GB:5.50,NVIDIA-L40S:1.80,NVIDIA-A10G:1.20,NVIDIA-T4:0.75
pricing.spotDiscountPercent Spot instance discount percentage 60

Optional integrations

ValueDescriptionDefault
prometheus.url Prometheus URL for DCGM and network metrics (empty)
watchNamespaces List of namespaces to watch (YAML array) [] (all)

Example Helm install

helm upgrade --install vbilling deploy/helm/vbilling \ --namespace vbilling-system --create-namespace \ --set lago.existingSecret="vbilling-lago-secret" \ --set billing.collectionInterval="120s" \ --set prometheus.url="http://prometheus.monitoring.svc.cluster.local:9090" \ --set pricing.cpuPerCoreHour="0.08" \ --set pricing.gpuDefaultPerHour="3.00"

Lago Pricing Configuration

vBilling bootstraps 9 billable metrics and a vcluster-standard plan on startup. All charges default to $0.00 per unit. You must configure pricing in the Lago UI or API to actually generate non-zero invoices.

Charges to configure

In the Lago UI, go to Plans > vCluster Standard > Edit and set the per-unit price for each charge:

Metric CodeAggregation FieldCharge ModelSuggested Price
vcluster_cpu_core_hours cpu_core_hours Standard (per unit) $0.05
vcluster_memory_gb_hours memory_gb_hours Standard (per unit) $0.01
vcluster_storage_gb_hours storage_gb_hours Standard (per unit) $0.001
vcluster_instance_hours instance_hours Standard (per unit) $0.10
vcluster_gpu_hours gpu_hours Standard (per unit) $2.50
vcluster_gpu_utilization gpu_util_score Standard (per unit) $0.00 (informational)
vcluster_network_egress_gb egress_gb Standard (per unit) $0.05
vcluster_lb_hours lb_hours Standard (per unit) $0.025
vcluster_private_node_hours private_node_hours Standard (per unit) $1.00

Example: Neocloud pricing

For a GPU-focused neocloud selling compute to AI teams, you might set higher GPU prices and include a premium for dedicated infrastructure:

# Neocloud pricing example CPU Core-Hours: $0.08 # higher due to bare-metal premium Memory GB-Hours: $0.015 # ECC memory cost Storage GB-Hours: $0.002 # NVMe SSD storage Instance Hours: $0.25 # base platform fee GPU Hours (default): $3.50 # A100 baseline GPU Hours (H100): $5.50 # premium for latest gen Network Egress/GB: $0.08 # cross-DC traffic LoadBalancer Hours: $0.05 # dedicated L4 LB Private Node Hours: $2.00 # full node reservation

Example: Internal platform team pricing

For an IDP (Internal Developer Platform) team doing internal chargebacks, use lower prices that reflect shared infrastructure costs:

# Internal platform team (chargeback) pricing CPU Core-Hours: $0.02 # internal cost only Memory GB-Hours: $0.005 # shared cluster memory Storage GB-Hours: $0.0005 # network-attached storage Instance Hours: $0.05 # minimal overhead fee GPU Hours (default): $1.00 # amortized GPU cost Network Egress/GB: $0.01 # internal network LoadBalancer Hours: $0.01 # shared ingress Private Node Hours: $0.50 # bare-metal amortization
!
If you see usage events in Lago but invoices show $0.00, it means the charge amounts are still set to their defaults ($0). Edit the plan charges in the Lago UI.
